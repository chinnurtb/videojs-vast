videojs-vast
============

VideoJS plugin for VAST XML.


## Testing

Install dev dependencies with NPM:

```bash
$ npm install -d
```

### Selenium

If `selenium` is not installed, running it from node_modules suffices:

```bash
$ node_modules/selenium/bin/selenium
```

### Chrome Driver

Download the latest Chrome Driver and make sure `$ chromedriver` is accessible within the PATH:

https://code.google.com/p/selenium/downloads/list

### Web server

Start a web server so that the examples folder is accessible.
